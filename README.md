# Market Trend Analytics Dashboard

## Overview
An interactive analytics dashboard built using Python and Streamlit to visualize near real-time market data. The dashboard enables users to track price trends, monitor key metrics, and derive insights through dynamic visualizations.

## Problem Statement
Analyzing market trends in real-time is essential for making informed decisions. This project focuses on building a system to monitor price movements and identify patterns using data-driven visualization.

## Features
- Near real-time data fetching using API-based sources  
- Interactive line and candlestick charts  
- KPI tracking (price, percentage change)  
- Auto-refresh functionality  
- Market open/closed status detection  

## Tech Stack
Python, Streamlit, Pandas, Plotly, yfinance

## Key Insights
- Price trends help identify momentum in assets  
- Short-term fluctuations indicate volatility patterns  
- Continuous monitoring supports better timing decisions  

## Use Case
This dashboard can be used to monitor market behavior and support analytical decision-making using real-time data.
